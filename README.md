# STS-Rating
A method to rate chess engines using STS test suite.

### A. Requirements
#### Python version >= 3.7.3

### B. Command line
`python3 sts_rating.py -f "STS1-STS15_LAN_v3.epd" -e Stockfish.exe -t 1 -h 128 --getrating --log --contempt 50`
